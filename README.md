# Personal Portfolio — Akbar Fai

My personal portfolio website. Built to replace the "coming soon"
on my GitHub profile and to have a proper place to show my work.

Live: [akbarfai.dev](https://akbarfai.dev) ← update once deployed

---

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** — scroll animations
- **next-themes** — dark mode
- **react-hook-form + zod** — contact form validation

---

## Features

- Dark mode (locked, no toggle)
- Live GitHub stats via GitHub REST + GraphQL API
- Smooth scroll animations
- Fully responsive
- Projects, Learning Journey timeline, Contact section
